## 🚀 Setup & Installation

### For Developers

1. **Clone the repository**
```bash
git clone https://github.com/vishutdhar/dr-dhar-clinic.git
cd dr-dhar-clinic
```

2. **Open in browser**
```bash
open index.html  # Mac
# OR
start index.html # Windows
```

### For Updates

1. **Make your changes to** `index.html`
2. **Push to GitHub**
```bash
git add .
git commit -m "Description of changes"
git push
```
3. **Website auto-updates in 2-5 minutes**

## 🔧 Customization Guide

### To update clinic timings:
Look for the `timing-table` section in `index.html`

### To update services:
Find the `services-grid` section and modify service cards

### To change contact info:
Update phone numbers and email in the contact section

### To modify colors:
Edit the CSS variables in the `:root` section:
```css
:root {
    --primary: #1e5f8e;
    --secondary: #52b788;
    --accent: #f77f00;
}
```

### To update doctor photo:
Replace `doctor-photo.jpg` with new image (same filename)
